无特殊说明的，所有代码顶格写，且写一行隔一行

标题

  • 代码：#bp标题

  • 说明：bp代表空格，记得空格，#越多字越小

图片

  • 代码：![图片名字](图片url地址)

  • 说明：名字可为空，地址选新浪微博

分割线

  • 代码：***=---=___

  • 说明：独占一行

表格

  • 代码：|课程表|周六|周日|

  • 说明：不隔行，紧贴着第二行输入|-后按tab键

代码

  • 代码：`代码内容`

  • 说明：点是数字1前面的那个键~的点，不是单引号

代码块

  • 代码：bpbpbpbp=一个tab键

  • 说明：没啥说的

无序列表

  • 代码：*bp列表第一个

  • 说明：记得空格

有序列表

  • 代码：1.bp列表第一个

  • 说明：记得英文句号和空格，有序列表前面要有纯文本做前提
